# Frontend Setup

## Установка

```bash
cd frontend
npm install
```

## Настройка

Создайте файл `.env` в корне папки `frontend`:

```
REACT_APP_API_URL=http://localhost:8000
```

## Запуск

```bash
npm start
```

Приложение откроется на http://localhost:3000

## Backend

Backend должен быть запущен на порту 8000. Для запуска backend:

```bash
cd backend
python main.py
```
